# DDD example for Kotlin WebApplication

## How to generate project

```
$ curl https://start.spring.io/starter.zip -d type=gradle-project -d language=kotlin -d style=web,jooq,h2 -d packageName=ddd -d name=ddd -o ddd.zip
$ unzip ddd.zip
$ rm ddd.zip
```
